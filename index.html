Ejemplo 1 Suma sucesiva
El usuario ingresara el valor limite con el cual se sumaran los valores sucesivos desde 0 hasta este valor
Si ingresamos 4 ,se realizara la siguiente suma : SumaSucesiva = 1+2+3+4 = 10

Ejemplo 2 Armar texto
Ingresando 1 se imprimira el texto
Ingresando 2 se desplegara un input donde el usuario ingresara una frase que se concatenara a lo ingresado anteriormente
Ingresando otra tecla terminara el programa.
